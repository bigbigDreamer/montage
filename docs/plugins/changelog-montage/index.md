# changelog-montage

A log plugin for changeset.

## Usage

```bash
$ npm i @montagejs/changelog-montage
```

In `config.json` with `.changeset` `dir`.

```json
{
    "changelog": [
        "@montagejs/changelog-montage",
        {
            "repo": "bigbigDreamer/montage",
            "author": "bigbigDreamer"
        }
    ]
}
```
